How to add audio and video to a website

For Audio

  1. Download a audio if you don't have a audio file in your PC
  2. Use an audio tag and add the audio file by using src tag
  3. And now view on your website

For video

  1. Download a video if you don't have a video file in your PC
  2. Use an audio tag and add the audio file by using src tag
  3. And now view on your website
